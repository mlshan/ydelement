# ql1dtech

> A Vue.js project

## Install
```
npm install ydelelement -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import ydelelement from "ydelement"

Vue.use(ydelelement)

// or
import {
  ydTechsupport,
  // ...
} from 'ydelement'

Vue.component(ydTechsupport.name, ydTechsupport)
```
